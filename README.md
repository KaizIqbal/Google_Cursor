# Installation Using GIT
Follow Command

      $ git clone https://github.com/KaizIqbal/Google_Cursor.git
      $ cd Google_Cursor/
      $ sudo chmod +x build.sh
      $ ./build.sh
      $ sudo chmod +x ./Installer_Google.sh
      $ sudo ./Installer_Google.sh
